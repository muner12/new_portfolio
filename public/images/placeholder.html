Portfolio Placeholder Images

The portfolio application requires the following images to be placed in the public/images directory:

Required Images

  • profile.jpg - A profile/portrait photo (approx. 600x800px)
  • project1.jpg - Image for project 1 (approx. 800x450px)
  • project2.jpg - Image for project 2 (approx. 800x450px)
  • project3.jpg - Image for project 3 (approx. 800x450px)
  • project-placeholder.jpg - Default placeholder for projects (approx. 800x450px)
  • blog-placeholder.jpg - Default placeholder for blog posts (approx. 800x450px)
  • og-image.jpg - OpenGraph image for social sharing (1200x630px)

How to Add Images

You can add your own images to the public/images directory to replace these placeholders. Make sure to use the same filenames or update the references in the code.

Recommended Sources for Placeholder Images

  • Unsplash - Free high-quality stock photos
  • Placeholder.com - Simple placeholder images
  • Lorem Picsum - Placeholder images from Unsplash
